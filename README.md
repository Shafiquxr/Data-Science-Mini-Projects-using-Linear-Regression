# Data-Science-Mini-Projects-using-Linear-Regression
A collection of beginner-friendly Data Science mini projects using Python, Pandas, Matplotlib, and Scikit-learn. Includes Linear Regression models for Student Marks Prediction, Sales Prediction, and Banking Loan Prediction with data preparation, visualization, model training, and performance evaluation using R² Score.


This repository contains three beginner-friendly Data Science projects implemented in Python using **Pandas**, **Matplotlib**, and **Scikit-learn**. Each project demonstrates the complete Machine Learning workflow, including data preparation, model training, prediction, evaluation, and visualization.

## 🚀 Projects Included

### 1. 🎓 Student Marks Prediction

* Predicts student marks based on study hours.
* Uses Linear Regression to analyze the relationship between study time and academic performance.

### 2. 💰 Sales Prediction

* Predicts sales based on advertising budget.
* Demonstrates how marketing investment influences sales using Linear Regression.

### 3. 🏦 Banking Loan Prediction

* Predicts loan amount approval based on a customer's annual income.
* Shows how banking data can be analyzed using regression techniques.
